## A-B-Testing-Data-Analysis-Project

A/B Testing using Z-Test on marketing campaign data to compare conversion rates between Ad and PSA groups. Includes statistical hypothesis testing, Python analysis, and data visualization to evaluate campaign effectiveness.


## 📌 Project Overview

This project focuses on performing A/B Testing using a Z-Test for proportions to evaluate the effectiveness of two marketing campaigns: Ad Campaign vs PSA (Public Service Announcement).


The main objective is to determine whether the Ad campaign significantly improves user conversion rates compared to the PSA campaign.

## 📊 Dataset


The dataset used in this project is marketing_AB.csv taken from kaggle.


It contains user-level marketing experiment data collected from two groups:


Ad Group → Users exposed to advertisements

PSA Group → Users exposed to public service announcements
